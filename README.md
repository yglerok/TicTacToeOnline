# Tic Tac Toe Online | Онлайн крестики-нолики

Приложение для онлайн игры в крестики-нолики.

### Стэк

- cpp-httplib - http-сервер для статической html-страницы игры, WebSocket для обработки событий в реальном времени (ходы, подключения)
- nlohmann/json - обмен данными
- PostgreSQL - хранение истории игр
- Cmake - сборка проекта

### Планируемые работы

1) Игровая логика 
- класс игры - поле, проверка победы, проверка ничьей, управление очередью ходов
- менеджер игр - хранение всех активных игр, создание новых игр, возврат игры по id, удаление завершенных игр
2) HTTP-сервер (страница игры)
3) WebSocket
- при подключении игрока создается новая комната, либо он присоединяется к существующей (клиенту отправляется id игры и сторона X/O)
4) Frontend
- html-страница с полем
- JS: подключение к WebSocket, при клике отправка сообщения клиенту, при получении сообщения обновление доски, отображение статуса
5) Добавление комнат
- при входе на главную виден список комнат или возможность создать свою
- при создании комнаты получение кода
- при вводе кода - присоединение к игре
6) База данных
- при завершении игры - сохранение в БД
- эндпоинт /stats - рейтинг игроков (победы/поражения)
